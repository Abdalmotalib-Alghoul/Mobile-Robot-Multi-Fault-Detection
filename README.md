The data set is available at https://zenodo.org/records/19672705.

To run the simulation, copy these folders and files into ~/catkin_ws/src/stretch_ros/stretch_navigation, then in the terminal run the automate_data_collection_*.py file for the needed fault injection.
For example: ~/catkin_ws/src/stretch_ros/stretch_navigation/scripts$ python3 automate_data_collection_outlier.py
Note: All scripts are hardcoded paths; you have to edit the files/map path in each file with your own package path.
