To run the simulation, copy these folsers/files and put them in the ~/catkin_ws/src/stretch_ros and then in the terminal run the automate_data_collection_*.py file of the specific fault.
For example: ~/catkin_ws/src/stretch_ros/stretch_navigation/scripts$ python3 automate_data_collection_outlier.py
Note: You have to edit the files/map path in each file with your package path
